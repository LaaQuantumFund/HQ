## リスク一覧

| リスク | 影響 | 対策 |
|---|---|---|
| 4月中にトレードが安定しない | 退職判断を延期 | GenSpark での錬成を加速。目標未達でも無理に退職しない |
| 貯金200万に届かない | 退職判断を延期 | 給与＋トレード利益の合算で現実的な着地点を毎週確認 |
| VPS セットアップ遅延 | AI組織 Phase 1 が遅れる | Phase 1 の間にトレード錬成は並行して進む。インフラ遅延は事業停止にならない |
| GenSpark 無制限終了（年内） | 壁打ちツール喪失 | Claude Max（¥16,000〜32,000/月）に移行 or 代替調査 |
| 法人登記手続きの遅延 | Biz Phase 2 スケジュール圧迫 | 4月中に法人登記の書類準備を並行進行 |
| トレードで大きな損失 | 貯金目標・メンタルへの打撃 | リスク管理ルール厳守（1トレード1〜2%）。損失後は GenSpark でメンタル回復の壁打ち |
| AIエージェントのコスト超過 | Claude Pro 枠の圧迫 | Haiku を定型タスクに活用。Agent Teams は大型タスクのみ使用 |
| note 非公式 API 仕様変更 | Playwright スクリプト破損 | 代替として手動コピー投稿に切り替え。CTO が即対応 |
| Obsidian ナレッジの意図せぬ流出 | トレード知見の無断公開 | COS のみが読み取り可。CMO が直接アクセスする経路を持たない設計を維持 |
